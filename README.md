## Steps to run
1. Run `setup.sh` by doing `./setup.sh` to install dependencies.
2. Do `./run.sh` to start a development server, then go to `localhost:5000`.

Alternatively,
1. Run `setup.sh` by doing `./setup.sh` to install dependencies.
2. Run `./run_lan.sh <port>`, where `<port>` is replaced with a port number, to launch the website on your local network. You can then visit it with `http://<local ip>:<port>`.
